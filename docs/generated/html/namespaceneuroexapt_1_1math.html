计算张量的熵（信息量，Nat）。

参数:
    x: 输入张量。若 ``already_softmax=True`` 则视为概率分布；否则内部会调用 ``softmax``。
    dim: 取熵的维度。
    already_softmax: 指示 ``x`` 是否已是概率分布，避免重复 ``softmax`` 带来多余开销。
