Detailed Description

基于性能导向的神经元分裂器

提供多种分裂策略：
1. 梯度导向分裂 - 基于梯度信息选择分裂位置
2. 激活导向分裂 - 基于激活模式选择分裂位置
3. 混合策略 - 综合考虑多种因素
4. 信息导向分裂 - 基于信息论指标
